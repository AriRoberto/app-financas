# App de Finanças Pessoal e Familiar

Projeto full-stack inicial para controlar receitas e despesas da família (você + esposa), com dashboard explicativo e cadastro de lançamentos.

## O que já está pronto
- **Backend (Express)** com endpoints para membros da família, transações, dashboard e sugestões.
- **Frontend (React + Vite)** com visual detalhado, cards, barras por categoria, tabela de lançamentos e formulário para adicionar receitas/despesas.
- **Fluxo familiar**: você pode registrar gastos seus e da esposa, consolidar os dados e visualizar impacto por pessoa.

## Como usar
1. Rode backend e frontend (passo a passo em `docs/COMO_RODAR_LOCAL.md`).
2. No painel web, abra a seção **"Lançar receita/despesa"**.
3. Escolha o membro (Você/Esposa), tipo, categoria, descrição, valor e data.
4. Salve e veja os dados atualizarem em:
   - Cards de resumo
   - Resumo por membro
   - Barras por categoria
   - Tabela de últimos lançamentos
