# App de Finanças Pessoal e Familiar

Projeto inicial com estrutura completa de **backend** e **frontend** para um app de controle de finanças pessoal/familiar com acesso web (responsivo para mobile).

## Estrutura atual
- `backend/`: API REST com dados de dashboard e sugestões de economia.
- `frontend/`: interface React consumindo os endpoints da API.
- `docs/COMO_RODAR_LOCAL.md`: passo a passo para rodar localmente.

## Funcionalidades já disponíveis
- Endpoint de saúde da API.
- Endpoint de dashboard financeiro (receitas, despesas, saldo, taxa de poupança e categorias).
- Endpoint de sugestões de economia.
- Tela web com cards de resumo financeiro.
- Lista de categorias para explicar para onde o dinheiro está indo.
- Sugestões inteligentes de gastos.
